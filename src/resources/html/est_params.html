Estimated parameters & shocksÂ¶

This functionality provides an interface to specify estimated parameters. It also provides interface to estimate standard error of selected exogenous variable (temporary shocks). This is the same information as in estimated_params block of .mod file (see figure below).

For each estimated parameter user is able to specify the prior shape, the prior mean, the prior std., and optionally the prior upper and lower bounds.

Similarly, for each temporarily shock, user is able to specify prior shape, the prior mean, the prior std., and optionally the prior upper and lower bounds.

Prior lower and upper bounds are checked against acceptable values.

Initial values are loaded form estim_params_ structure if the .mod file already contains the information on estimated parameters.

User defined values are stored in in DYNARE structure estim_params_ and in Dynare_GUI structure model_settings.estim_params.
