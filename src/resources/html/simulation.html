Stochastic simulation¶

This functionality provides an interface to run stochastic simulation of the model with a specific interface for setting the options for DYNARE stoch_simul command (see figure below).

Model shocks and variables are grouped and displayed as defined in model settings. Only shocks and variables that have shown attribute set are displayed on this screen.

After user defines stoch_simul command options, simulation is performed. This is done with internal function call to DYNARE’ stoch_simul() function.
