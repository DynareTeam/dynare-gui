Shock decomposition¶

This functionality provides an interface for displaying decompositions in levels of the observed variables (see figures below).

On the left part of the screen model variables are grouped and displayed as defined in model settings. Only variables that have shown attribute set are displayed on this screen.

On the right part of the screen shock_decomposition command option parameter_set is displayed, together with historical period selection and option to use shock groups when displaying decomposition.

Default value for parameter set is posterior_mean if Metropolis has been run, else posterior_mode.

Shocks grouping functionality implements shock decomposition with grouping of structural shocks. Shocks are grouped as defined in model settings.

Run shock decomposition screen

Shock decomposition is done with internal function call to new DYNARE’ shock_decomposition() function which displays default decomposition screen together with individual plots (see figures below).
