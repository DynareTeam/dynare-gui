Installation and configuration
============

Dynare_GUI  works on Windows platform with Matlab versions from v.R2014b and newer.

To install and run Dynare_GUI:

#. Unzip Dynare_GUI source code to local folder (for example c:\\dynare\\dynare-gui)

#. Temporarly add the dynare/matlab folder to the Matlab's path:
	``>> addpath c:\dynare\4.x.y\matlab``

#. Temporarly add folders containing Dynare_GUI source code to the Matlab's path:
	``>> addpath c:\dynare\dynare-gui\src``
	
	``>> addpath c:\dynare\dynare-gui\src\resources``


#. Launch Dynare_GUI:
	``>> dynare_gui``
