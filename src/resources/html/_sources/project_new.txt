.. _project_new:

Create new project
============

This functionality allows user to create new project. User is displayed new project form (see Figure below) which contains following information:
	* project name (required field)
	* project folder (required field)
	* project description
	* project specific settings:
		* model type
		* latex representation of the model
		* maximum forecast periods.
