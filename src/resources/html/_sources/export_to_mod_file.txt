Export to .mod file
============

This functionality creates new .dyn file based on the current settings in the Dynare_GUI . It can be used with @#include directive in some future ‘.mod’ file.

User is asked to specify location and name of the .dyn file with standard file browsing interface.

Generated .dyn file includes following parts:
	* vars, params, shocks
	* estimated_params
	* varobs
	* DYNARE commands used with Dynare_GUI :
		* simul command (deterministic simulation)
		* stoch_simul command (stochastic simulation)
		* calib_smoother command
		* estimation command
		* conditional_forecast command.
