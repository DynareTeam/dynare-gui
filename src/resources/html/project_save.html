This functionality allows saving of all project related variables in project (.dproj) file.

This functionality also allows modification of all project related information except for project name (this is part of Save Project As functionality).

When project is saved current versions of all project related variables (structures) are stored in .dproj file. Whenever project is saved, user is asked to confirm if he/she wants to override existing project.
